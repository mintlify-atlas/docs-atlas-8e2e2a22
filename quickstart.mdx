## Prerequisites

- Python 3.8 or later
- An audio file to test with (WAV, MP3, FLAC, or any format your audio backend supports)

## Steps

<Steps>
  <Step title="Install silero-vad">
    Install the package from PyPI. This pulls in PyTorch and torchaudio automatically.

    ```bash
    pip install silero-vad
    ```

    <Tip>
      To run inference with the ONNX backend instead of PyTorch JIT, install the optional ONNX extras. See the [installation page](/installation) for all available options.
    </Tip>
  </Step>

  <Step title="Load the model">
    Import `load_silero_vad` and call it to load the bundled JIT model. The model is stored inside the package â€” no download required.

    ```python
    from silero_vad import load_silero_vad

    model = load_silero_vad()
    ```

    To load the ONNX model instead, pass `onnx=True`:

    ```python
    model = load_silero_vad(onnx=True)
    ```
  </Step>

  <Step title="Read an audio file">
    Use `read_audio` to load an audio file and resample it to 16 kHz. The function returns a 1-D `torch.Tensor` of float32 samples.

    ```python
    from silero_vad import read_audio

    wav = read_audio('path/to/audio.wav')
    ```

    `read_audio` accepts an optional `sampling_rate` argument. The default is `16000`. Pass `sampling_rate=8000` if you want 8 kHz inference instead.
  </Step>

  <Step title="Get speech timestamps">
    Pass the waveform and model to `get_speech_timestamps`. Set `return_seconds=True` to receive timestamps in seconds rather than sample indices.

    ```python
    from silero_vad import get_speech_timestamps

    speech_timestamps = get_speech_timestamps(wav, model, return_seconds=True)

    for segment in speech_timestamps:
        print(f"Speech from {segment['start']:.1f}s to {segment['end']:.1f}s")
    ```

    `get_speech_timestamps` returns a list of dicts. Each dict has two keys:

    | Key | Type | Description |
    | --- | --- | --- |
    | `start` | `float` or `int` | Start of the speech segment. Seconds when `return_seconds=True`, otherwise sample index. |
    | `end` | `float` or `int` | End of the speech segment. Seconds when `return_seconds=True`, otherwise sample index. |
  </Step>
</Steps>

## Complete example

<CodeGroup>

```python pip install
from silero_vad import load_silero_vad, read_audio, get_speech_timestamps

# Load model (PyTorch JIT, bundled â€” no download needed)
model = load_silero_vad()

# Read audio and resample to 16 kHz
wav = read_audio('path/to/audio.wav')

# Detect speech segments
speech_timestamps = get_speech_timestamps(wav, model, return_seconds=True)

for segment in speech_timestamps:
    print(f"Speech from {segment['start']:.1f}s to {segment['end']:.1f}s")
```

```python torch.hub
import torch
torch.set_num_threads(1)

# Load model and utilities directly from the GitHub repo (no pip install needed)
model, utils = torch.hub.load(repo_or_dir='snakers4/silero-vad', model='silero_vad')

# Detect speech segments

for segment in speech_timestamps:
    print(f"Speech from {segment['start']:.1f}s to {segment['end']:.1f}s")
```

## Working with the returned timestamps

Once you have `speech_timestamps`, you can extract or discard speech segments from the original waveform using the `collect_chunks` and `drop_chunks` utilities.

```python
from silero_vad import (
    load_silero_vad,
    read_audio,
    get_speech_timestamps,
    collect_chunks,
    drop_chunks,
)

model = load_silero_vad()
wav = read_audio('path/to/audio.wav')

# Get timestamps in samples (default â€” no return_seconds)
speech_timestamps = get_speech_timestamps(wav, model)

# Concatenate only the speech portions
speech_only = collect_chunks(speech_timestamps, wav)

# Concatenate only the non-speech portions
silence_only = drop_chunks(speech_timestamps, wav)
```

<Tip>
  When using `collect_chunks` or `drop_chunks` with second-based timestamps (i.e., `return_seconds=True`), pass `seconds=True` and `sampling_rate=16000` to those functions as well.
</Tip>

## Key parameters

`get_speech_timestamps` exposes several tunable parameters. The most useful ones for getting started:

| Parameter | Default | Description |
| --- | --- | --- |
| `threshold` | `0.5` | Speech probability cutoff. Lower values detect more speech; higher values are stricter. |
| `sampling_rate` | `16000` | Must match the sampling rate of the waveform. Use `8000` or `16000`. |
| `min_speech_duration_ms` | `250` | Speech segments shorter than this (ms) are discarded. |
| `min_silence_duration_ms` | `100` | Minimum silence (ms) between separate speech segments. |
| `return_seconds` | `False` | Return timestamps in seconds instead of sample indices. |
